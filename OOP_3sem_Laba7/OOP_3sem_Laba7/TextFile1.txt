 1. Что такое обобщение (generic)?
Обобщение (или шаблон) — это механизм в C, позволяющий создавать классы, методы и интерфейсы, которые могут работать с любым типом данных.
Это позволяет создавать более гибкий и повторно используемый код, избегая избыточности,
связанной с созданием разных версий одного и того же класса или метода для различных типов.

3. Как можно наложить определенное ограничение на параметр?
Ограничение на параметр в обобщении можно наложить с помощью ключевого слова where. Например:

class Test<T> where T : class
{
}

4. Как можно наложить несколько ограничений на параметр?
Несколько ограничений можно наложить, используя ключевое слово where для каждого ограничения. Например:

class Test<T> where T : class, new()
{
}

5. Перечислите все существующие ограничения на типы данных обобщения?
1. class — ограничение на ссылочные типы.
2. struct — ограничение на значимые типы.
3. new() — требует, чтобы тип имел конструктор без параметров.
4. Типы, производные от других типов (например, where T : BaseClass).
5. Интерфейсы (например, where T : IMyInterface).

9. Приведите примеры, когда обобщенный класс может действовать как базовый или производный класс.
Обобщенный класс может действовать как базовый или производный класс, когда он наследует другой обобщенный класс или реализует обобщенный интерфейс. Например:

class Base<T> { }
class Derived<T> : Base<T> { }


10. В каких случаях в обобщениях может использоваться оператор default?
Оператор default может использоваться для получения значения по умолчанию для типа параметра обобщения. Например:

T value = default(T);

Если T является ссылочным типом, default(T) вернет null. Если T — значимый тип, вернется значение по умолчанию для этого типа (например, 0 для int).

11. Поясните как использовать статические переменные в обобщенных классах.
В C нельзя использовать статические переменные типа параметров обобщений. Однако можно использовать статические переменные, определенные на уровне класса. Например:

class Test<T>
{
    public static int Count { get; set; }
}

В этом примере Count будет статической переменной для каждого типа T, но ее нельзя будет использовать для хранения информации о каждом конкретном T.

13. В чем отличие обобщенных классов от обобщенных структур?
Основное отличие состоит в том, что обобщенные классы могут наследоваться, а обобщенные структуры — нет.
Обобщенные классы могут быть созданы как ссылочные типы, в то время как обобщенные структуры являются значимыми типами.
Обобщенные структуры не могут содержать null, тогда как классы могут.

14. Какие классы для работы с файлами вы знаете? Приведите пример.
В C для работы с файлами можно использовать следующие классы:
1. File — предоставляет статические методы для создания, копирования, перемещения и удаления файлов.
2. FileInfo — предоставляет экземплярные методы для работы с файлами.
3. StreamReader и StreamWriter — используются для чтения и записи текстовых файлов.
